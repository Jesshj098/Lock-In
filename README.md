# Lock In

## Project Overview

Lock In is a productivity application designed to help users stay focused during study or work sessions.

The main purpose of the project is to reduce distractions caused by social media or other non-productive applications while the user is working. The system monitors the applications running on the computer and detects when the user opens apps considered distracting.

When a distraction is detected, the program sends a notification reminding the user to return to their task and stay focused.

This project aims to encourage better study habits, improve time management, and increase productivity through simple but effective focus tracking.

## Main Objectives

- Help users maintain focus during work or study sessions
- Detect distracting applications automatically
- Provide reminders to reduce distractions
- Promote productivity and discipline

## Motivation

Nowadays, distractions from phones, social media, and entertainment apps make it difficult to stay concentrated for long periods. Lock In was created as a solution to help users build better focus habits and improve efficiency.
